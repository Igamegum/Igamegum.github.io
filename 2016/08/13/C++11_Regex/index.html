C++11正则表达式

  • C++
  • 正则表达式

上班的时候抽空学了一下 C++ 正则表达式的使用方法,在此作个小结！

编译需加选项 -std=c++11

1.std::regex_match

参考文档

   1.1std::regex_match(std::string,std::regex)

     string 成功 匹配 regex 返回 true,其他情况返回false

   1.2std::regex_match(std::string,std::match_results,std::regex)

     首先整个string要匹配regex,其次std::match_results中依次存储regex中的子表达式的匹配子串(空串也是子串)

输出结果

12345
123
 
45

2.std::regex_search

参考文档

   2.1std::regex_search(std::string,std::match_results,std::regex)

     regex_search 只要string中存在符合regex的子串就返回结果，而regex_match要求整个string符合regex

输出结果

12345 123 45
05486 054 86

3.std::regex_replace

参考文档

   3.1std::regex_replace(std::string,std::regex,std::string)

     将string1中，符合regex的子串用string2替换

     代码中replacer的 $2 代表 regex中的第二个子表达式

输出结果

WeMadeFox

   3.2std::regex_replace(std::iterator,std::iterator,std::iterator,std::regex,std::string)

     与上面一个函数的区别它支持范围性的匹配，并且能将匹配结果存储到迭代器指向的内存中

输出结果

MadeFox
